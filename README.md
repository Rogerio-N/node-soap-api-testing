# Soap API Testing with Node

![Node](https://img.shields.io/badge/Node-5FA04E?style=for-the-badge&logo=nodedotjs&logoColor=white)
![JavaScript](https://img.shields.io/badge/javascript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)
![Pnpn](https://img.shields.io/badge/pnpn-F69220?style=for-the-badge&logo=pnpm&logoColor=white)

This is an api testing automation project, that tests the [CountryInfoService](http://webservices.oorsprong.org/websamples.countryinfo/CountryInfoService.wso) (a demo soap api). The porpouse of this project is demonstrate the the execution of api requests using the SOAP Protocol and the Node.js capabilities to test thoses requests with minimal setup and external dependencies.

### Pre-requisites

- [Node 24](https://nodejs.org/en/download/)
- [PNPN](https://pnpm.io/pt/)

### Setup

1 - Install project dependencies: 
```
pn install
```

2 - To run the tests: 
```
pn test
```
